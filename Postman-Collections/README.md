This folder contains my Postman collection and API testing documentation.

- Telephone Book Service API Tests
This repository contains a Postman collection showcasing a comprehensive set of tests I designed for a Telephone Book Service API. The tests were developed as part of a coding challenge, with a focus on validating and extending the functionality of a pre-existing REST API service. While the service code itself cannot be shared due to ownership restrictions, this collection demonstrates my API testing skills and approach to quality assurance.

The Telephone Book Service is a small REST API application for managing contact data. Its primary features include:

  Adding, retrieving, updating, and deleting contacts.
  Searching for contacts by first name, last name, or phone number.
  During the coding test, additional features were required, such as:

  Implementing missing REST methods for update and delete operations.
  Enhancing search functionality to search across multiple fields simultaneously in a case-insensitive manner.
  Improving error handling for non-existent resources.
